**Step 4: Create `writ-compiler/src/ast/types.rs`**

Define `AstType` â€” the lowered type representation. Key decisions:

- NO `Nullable` variant â€” `T?` lowers to `Generic { name: "Option", args: [T] }` before reaching AST
- `Named` covers simple types: `int`, `string`, `Guard`, etc.
- `Generic` covers parameterized types: `Option<T>`, `List<T>`, `Result<A, B>`
- `Array` covers `T[]`
- `Func` covers function types: `fn(int, string) -> bool`
- `Void` is explicit

Every variant carries `span: SimpleSpan`.

```rust
use chumsky::span::SimpleSpan;

#[derive(Debug, Clone, PartialEq)]
pub enum AstType {
    /// Named type: `int`, `string`, `Guard`
    Named { name: String, span: SimpleSpan },
    /// Generic type: `Option<T>`, `List<T>`, `Result<A, B>`
    /// NOTE: T? lowers to Generic { name: "Option", args: [T] } â€” no Nullable variant.
    Generic { name: String, args: Vec<AstType>, span: SimpleSpan },
    /// Array type: `T[]`
    Array { elem: Box<AstType>, span: SimpleSpan },
    /// Function type: `fn(int, string) -> bool`
    Func { params: Vec<AstType>, ret: Option<Box<AstType>>, span: SimpleSpan },
    /// Void type
    Void { span: SimpleSpan },
}
```

**Step 5: Create `writ-compiler/src/ast/expr.rs`**

Define `AstExpr` â€” all expression forms that survive lowering. Key decisions:

- NO `FormattableString` â€” lowered to string concatenation chains before reaching AST
- NO compound `Assign` variants â€” `+=` lowered to `a = a + b` before reaching AST
- NO `NullLit` â€” lowered to `Option::None` (a path expression) before reaching AST
- YES `Spawn`, `Join`, `Cancel`, `Defer`, `Detached` â€” concurrency pass-through (R1)
- YES `Error` â€” error recovery sentinel (R1)
- `Assign` is plain assignment only (`=`), no compound operators

Include supporting types: `BinaryOp`, `PrefixOp`, `PostfixOp`, `RangeKind`, `AstArg`, `AstLambdaParam`, `AstMatchArm`, `AstPattern`.

Derive from CST `Expr` variants but with owned data (`String` not `&'src str`) and span-per-variant.

```rust
use chumsky::span::SimpleSpan;
use crate::ast::stmt::AstStmt;
use crate::ast::types::AstType;

#[derive(Debug, Clone, PartialEq)]
pub enum AstExpr {
    // Literals
    IntLit { value: i64, span: SimpleSpan },
    FloatLit { value: f64, span: SimpleSpan },
    StringLit { value: String, span: SimpleSpan },
    BoolLit { value: bool, span: SimpleSpan },
    SelfLit { span: SimpleSpan },
    // NOTE: No NullLit â€” lowered to path expression Option::None before reaching AST

    // Identifiers and paths
    Ident { name: String, span: SimpleSpan },
    Path { segments: Vec<String>, span: SimpleSpan },

    // Binary and unary operations
    Binary { left: Box<AstExpr>, op: BinaryOp, right: Box<AstExpr>, span: SimpleSpan },
    UnaryPrefix { op: PrefixOp, expr: Box<AstExpr>, span: SimpleSpan },
    UnaryPostfix { expr: Box<AstExpr>, op: PostfixOp, span: SimpleSpan },

    // Access
    MemberAccess { object: Box<AstExpr>, field: String, field_span: SimpleSpan, span: SimpleSpan },
    BracketAccess { object: Box<AstExpr>, index: Box<AstExpr>, span: SimpleSpan },

    // Calls
    Call { callee: Box<AstExpr>, args: Vec<AstArg>, span: SimpleSpan },
    GenericCall { callee: Box<AstExpr>, type_args: Vec<AstType>, args: Vec<AstArg>, span: SimpleSpan },

    // Control flow (expression forms)
    If { condition: Box<AstExpr>, then_block: Vec<AstStmt>, else_block: Option<Box<AstExpr>>, span: SimpleSpan },
    IfLet { pattern: Box<AstPattern>, value: Box<AstExpr>, then_block: Vec<AstStmt>, else_block: Option<Box<AstExpr>>, span: SimpleSpan },
    Match { scrutinee: Box<AstExpr>, arms: Vec<AstMatchArm>, span: SimpleSpan },
    Block { stmts: Vec<AstStmt>, span: SimpleSpan },

    // Range
    Range { start: Option<Box<AstExpr>>, kind: RangeKind, end: Option<Box<AstExpr>>, span: SimpleSpan },
    FromEnd { expr: Box<AstExpr>, span: SimpleSpan },

    // Lambda
    Lambda { params: Vec<AstLambdaParam>, return_type: Option<Box<AstType>>, body: Vec<AstStmt>, span: SimpleSpan },

    // Concurrency pass-through (R1: first-class AST nodes)
    Spawn { expr: Box<AstExpr>, span: SimpleSpan },
    Detached { expr: Box<AstExpr>, span: SimpleSpan },
    Join { expr: Box<AstExpr>, span: SimpleSpan },
    Cancel { expr: Box<AstExpr>, span: SimpleSpan },
    Defer { expr: Box<AstExpr>, span: SimpleSpan },
    Try { expr: Box<AstExpr>, span: SimpleSpan },

    // Array literal
    ArrayLit { elements: Vec<AstExpr>, span: SimpleSpan },

    // Plain assignment only (compound assignments lowered away before reaching AST)
    Assign { target: Box<AstExpr>, value: Box<AstExpr>, span: SimpleSpan },

    // Error recovery sentinel
    Error { span: SimpleSpan },
}

// --- Supporting types ---

#[derive(Debug, Clone, PartialEq)]
pub enum BinaryOp {
    Add, Sub, Mul, Div, Mod,
    Eq, NotEq, Lt, Gt, LtEq, GtEq,
    And, Or,
    BitAnd, BitOr,
}

#[derive(Debug, Clone, PartialEq)]
pub enum PrefixOp {
    Neg,
    Not,
    FromEnd,
}

#[derive(Debug, Clone, PartialEq)]
pub enum PostfixOp {
    /// `?` null propagation â€” early return on None
    NullPropagate,
    /// `!` unwrap â€” panic on None
    Unwrap,
}

#[derive(Debug, Clone, PartialEq)]
pub enum RangeKind {
    Exclusive,
    Inclusive,
}

#[derive(Debug, Clone, PartialEq)]
pub struct AstArg {
    pub name: Option<String>,
    pub value: AstExpr,
    pub span: SimpleSpan,
}

#[derive(Debug, Clone, PartialEq)]
pub struct AstLambdaParam {
    pub name: String,
    pub ty: Option<AstType>,
    pub span: SimpleSpan,
}

#[derive(Debug, Clone, PartialEq)]
pub struct AstMatchArm {
    pub pattern: AstPattern,
    pub body: Vec<AstStmt>,
    pub span: SimpleSpan,
}

#[derive(Debug, Clone, PartialEq)]
pub enum AstPattern {
    Literal { expr: Box<AstExpr>, span: SimpleSpan },
    Wildcard { span: SimpleSpan },
    Variable { name: String, span: SimpleSpan },
    EnumDestructure { path: Vec<String>, fields: Vec<AstPattern>, span: SimpleSpan },
    Or { patterns: Vec<AstPattern>, span: SimpleSpan },
    Range { start: Box<AstExpr>, kind: RangeKind, end: Box<AstExpr>, span: SimpleSpan },
}
```

**Step 6: Create `writ-compiler/src/ast/stmt.rs`**

Define `AstStmt` â€” all statement forms. Key decisions:

- NO `DlgDecl` variant â€” dialogue is lowered to `Fn` before reaching the AST
- NO `Transition` variant â€” dialogue transitions lower to `Return` statements
- `Atomic` survives as-is
- YES `Error` variant for error recovery (R1)

```rust
use chumsky::span::SimpleSpan;
use crate::ast::expr::AstExpr;
use crate::ast::types::AstType;

#[derive(Debug, Clone, PartialEq)]
pub enum AstStmt {
    /// Variable declaration: `let [mut] name [: type] = expr;`
    Let { mutable: bool, name: String, name_span: SimpleSpan, ty: Option<AstType>, value: AstExpr, span: SimpleSpan },
    /// Expression statement: `expr;`
    Expr { expr: AstExpr, span: SimpleSpan },
    /// For loop: `for name in expr { body }`
    For { binding: String, binding_span: SimpleSpan, iterable: AstExpr, body: Vec<AstStmt>, span: SimpleSpan },
    /// While loop: `while condition { body }`
    While { condition: AstExpr, body: Vec<AstStmt>, span: SimpleSpan },
    /// Break: `break [expr]`
    Break { value: Option<AstExpr>, span: SimpleSpan },
    /// Continue
    Continue { span: SimpleSpan },
    /// Return: `return [expr]`
    Return { value: Option<AstExpr>, span: SimpleSpan },
    /// Atomic block: `atomic { body }`
    Atomic { body: Vec<AstStmt>, span: SimpleSpan },
    /// Error recovery sentinel
    Error { span: SimpleSpan },
}
```

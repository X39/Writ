```rust
fn lower_src(src: &'static str) -> Ast {
    let (items, parse_errors) = writ_parser::parse(src);
    let items = items.expect("parse returned None");
    let error_msgs: Vec<String> = parse_errors.iter().map(|e| format!("{e:?}")).collect();
    assert!(error_msgs.is_empty(), "parse errors: {:?}", error_msgs);
    let (ast, lower_errors) = lower(items);
    assert!(lower_errors.is_empty(), "lowering errors: {:?}", lower_errors);
    ast
}
```

**Test 2: `concurrency_join_passthrough`**
- Source: `"fn f() { join h; }"`
- Verifies: `AstExpr::Join { expr: AstExpr::Ident { name: "h" } }`
- Note: If `join h;` doesn't parse as an expression statement, try `"fn f(h: Handle) { join h; }"` or adjust syntax per parser.

**Test 3: `concurrency_cancel_passthrough`**
- Source: `"fn f() { cancel h; }"`
- Verifies: `AstExpr::Cancel { expr: AstExpr::Ident { name: "h" } }`

**Test 4: `concurrency_defer_passthrough`**
- Source: `"fn f() { defer cleanup(); }"`
- Verifies: `AstExpr::Defer { expr: AstExpr::Call { callee: Ident("cleanup") } }`
- Note: The spec shows `defer { ... }` block syntax but the CST has `Expr::Defer(Box<Spanned<Expr>>)` wrapping any expression. Try both `"fn f() { defer cleanup(); }"` and `"fn f() { defer { cleanup(); } }"` — use whichever parses.

**Test 5: `concurrency_detached_spawn_passthrough`**
- Source: `"fn f() { spawn detached doWork(); }"`
- Verifies: `AstExpr::Spawn { expr: AstExpr::Detached { expr: AstExpr::Call { callee: Ident("doWork") } } }`
- Note: `spawn detached expr` parses as `Spawn(Detached(Call(...)))` — nested, not flat. The snapshot should show this nesting.

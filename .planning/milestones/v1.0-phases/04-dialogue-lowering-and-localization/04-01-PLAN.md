**3. Public entry point: `lower_dialogue`**

```rust
pub fn lower_dialogue(
    dlg: DlgDecl<'_>,
    dlg_span: SimpleSpan,
    ctx: &mut LoweringContext,
) -> AstFnDecl
```

Implementation steps:
1. Extract dialogue name: `let dlg_name = dlg.name.0.to_string();`
2. Lower params: `let params = dlg.params.unwrap_or_default().into_iter().map(|(p, ps)| lower_param(p, ps)).collect::<Vec<_>>();`
3. Collect param names for Tier 1 speaker lookup: `let param_names: Vec<String> = params.iter().map(|p| p.name.clone()).collect();`
4. **Pre-scan for singleton speakers (Tier 2 hoisting):** Call `collect_singleton_speakers(&dlg.body, &param_names)` to get `Vec<(String, SimpleSpan)>` — deduplicated list of speaker names that are NOT params (these need `Entity.getOrCreate<T>()` hoisting).
5. **Generate hoisted let-bindings:** For each `(speaker_name, speaker_span)` in the singleton list, create:
   ```rust
   AstStmt::Let {
       mutable: false,
       name: format!("_{}", speaker_name.to_lowercase()),
       name_span: speaker_span,
       ty: None,
       value: AstExpr::GenericCall {
           callee: Box::new(AstExpr::MemberAccess {
               object: Box::new(AstExpr::Ident { name: "Entity".to_string(), span: speaker_span }),
               field: "getOrCreate".to_string(),
               field_span: speaker_span,
               span: speaker_span,
           }),
           type_args: vec![AstType::Named { name: speaker_name.clone(), span: speaker_span }],
           args: vec![],
           span: speaker_span,
       },
       span: speaker_span,
   }
   ```
6. **Initialize dialogue lowering state:** Create a `DlgLowerState` struct (private to the module) to track:
   - `dlg_name: String` — for FNV-1a namespace
   - `namespace: String` — use empty string `""` for Phase 4 (namespace context not yet threaded — see Research Open Question #2)
   - `param_names: Vec<String>` — for Tier 1 lookup
   - `singleton_names: Vec<String>` — for Tier 2 speaker refs
   - `occurrence_tracker: HashMap<(String, String, String, String), u32>` — for FNV-1a dedup index `(namespace, method, speaker, content) → count`
   - `manual_keys: HashMap<String, SimpleSpan>` — for `#key` collision detection
7. **Lower dialogue body:** Call `lower_dlg_lines(&dlg.body, &mut state, ctx)` → `Vec<AstStmt>`
8. **Combine:** hoisted let-bindings + lowered body statements
9. Return `AstFnDecl { attrs: vec![], vis: None, name: dlg_name, name_span: dlg.name.1, generics: vec![], params, return_type: None, body: combined_stmts, span: dlg_span }`

**5. Private: `lower_dlg_lines`**

```rust
fn lower_dlg_lines(
    lines: &Vec<Spanned<DlgLine<'_>>>,
    state: &mut DlgLowerState,
    ctx: &mut LoweringContext,
) -> Vec<AstStmt>
```

Iterate over lines with index. For each `(line, line_span)`:

- **`DlgLine::SpeakerLine { speaker, text, loc_key }`:**
  1. Resolve speaker ref (see `resolve_speaker` below)
  2. Build raw text content string (concatenating `DlgTextSegment::Text` parts with `{expr}` placeholders preserved literally for key computation)
  3. Compute or use manual loc key (see `compute_or_use_loc_key` below)
  4. Lower text segments to fallback expression (see `lower_dlg_text`)
  5. Emit `AstStmt::Expr { expr: make_say_localized(speaker_ref, key, fallback, line_span), span: line_span }`

- **`DlgLine::SpeakerTag((speaker_name, speaker_span))`:**
  1. Push speaker scope: `ctx.push_speaker(SpeakerScope { name: speaker_name.to_string(), span: speaker_span })`
  2. Emit NO statement — side effect only

- **`DlgLine::TextLine { text, loc_key }`:**
  1. Get current speaker via `ctx.current_speaker()`. If `None`, emit `LoweringError::UnknownSpeaker { name: "".to_string(), span: line_span }` and use `AstExpr::Error { span: line_span }` as speaker ref. If `Some`, resolve speaker ref using `resolve_speaker` with the current speaker's name.
  2. Same loc key + text lowering as SpeakerLine

- **`DlgLine::CodeEscape((escape, escape_span))`:**
  Match on escape:
  - `DlgEscape::Statement(stmt)` → `vec![lower_stmt(*stmt, ctx)]`
  - `DlgEscape::Block(stmts)` → `stmts.into_iter().map(|s| lower_stmt(s, ctx)).collect()`

- **`DlgLine::Choice((choice, choice_span))`:**
  Call `lower_choice(choice, choice_span, state, ctx)` → `AstStmt::Expr`

- **`DlgLine::If((dlg_if, if_span))`:**
  Call `lower_dlg_if(dlg_if, if_span, state, ctx)` → `AstStmt::Expr`

- **`DlgLine::Match((dlg_match, match_span))`:**
  Call `lower_dlg_match(dlg_match, match_span, state, ctx)` → `AstStmt::Expr`

- **`DlgLine::Transition((trans, trans_span))`:**
  1. **Non-terminal check:** If this is NOT the last line (`i < lines.len() - 1`), emit `LoweringError::NonTerminalTransition { span: trans_span }`
  2. Call `lower_transition(trans, trans_span, ctx)` → `AstStmt::Return`

**8. Private: `fnv1a_32`**

```rust
fn fnv1a_32(input: &str) -> String {
    const OFFSET_BASIS: u32 = 0x811c9dc5;
    const PRIME: u32 = 0x01000193;
    let mut hash: u32 = OFFSET_BASIS;
    for &byte in input.as_bytes() {
        hash ^= byte as u32;
        hash = hash.wrapping_mul(PRIME);
    }
    format!("{:08x}", hash)
}
```

Per spec section 25.2.2 — exact algorithm mandated. Do NOT use an external crate.

**12. Private: `lower_choice`**

```rust
fn lower_choice(
    choice: DlgChoice<'_>,
    choice_span: SimpleSpan,
    state: &mut DlgLowerState,
    ctx: &mut LoweringContext,
) -> AstStmt
```

For each `(arm, arm_span)` in `choice.arms`:
1. **Save speaker scope depth:** `let depth = ctx.speaker_stack_depth();`
2. **Compute loc key for choice label:** Use `compute_or_use_loc_key` with speaker `""` (empty string for choice labels per spec section 25.2.1) and the label text as content.
3. **Lower arm body:** `let body = lower_dlg_lines(&arm.body, state, ctx);`
4. **Restore speaker scope:** Pop back to saved depth:
   ```rust
   while ctx.speaker_stack_depth() > depth {
       ctx.pop_speaker();
   }
   ```

**13. Private: `lower_dlg_if`**

```rust
fn lower_dlg_if(
    dlg_if: DlgIf<'_>,
    if_span: SimpleSpan,
    state: &mut DlgLowerState,
    ctx: &mut LoweringContext,
) -> AstStmt
```

Lowers to:
```rust

    expr: AstExpr::If {
        condition: Box::new(lower_expr(*dlg_if.condition, ctx)),
        then_block: lower_dlg_lines(&dlg_if.then_block, state, ctx),
        else_block: match dlg_if.else_block {
            None => None,
            Some(box (DlgElse::ElseIf(elif), elif_span)) => {
                // Recursive: lower_dlg_if(elif, elif_span, state, ctx) → wrap in Block
                Some(Box::new(AstExpr::Block {
                    stmts: vec![lower_dlg_if(elif, elif_span, state, ctx)],
                    span: elif_span,
                }))
            }
            Some(box (DlgElse::Else(lines), else_span)) => {
                Some(Box::new(AstExpr::Block {
                    stmts: lower_dlg_lines(&lines, state, ctx),
                    span: else_span,
                }))
            }
        },
        span: if_span,
    },
    span: if_span,
}
```

**Important:** The `else_block` field on `AstExpr::If` takes `Option<Box<AstExpr>>` (not `Vec<AstStmt>`). Wrap the else-branch stmts in an `AstExpr::Block`. For chained `else if`, recursively lower the inner `DlgIf` and wrap in a `Block` containing that single statement.

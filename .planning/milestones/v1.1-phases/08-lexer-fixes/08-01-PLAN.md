  <behavior>
    - raw_string callback: after counting extra quotes, verify the byte immediately after all opening quotes is `\n` (or `\r\n`). If not, return false (Error token).
    - raw_string callback: when the closing delimiter is found, verify that all characters on that line before the closing quotes are whitespace-only. Specifically: scan backwards from the closing quotes to the preceding newline (or start of string) and check that everything in between is spaces/tabs only. If non-whitespace precedes the closing delimiter on its line, return false.
    - formattable_raw_string callback: apply the same two validations (newline after opening, whitespace-only before closing).
    - formattable_string callback: in the `b'\\'` match arm, after advancing past `\\`, check if the next byte is `u` AND the byte after that is `{`. If so, skip past the `{` and scan until `}` is found (or end of string), consuming the entire `\u{...}` sequence without incrementing brace_depth. This prevents `\u{XXXX}` from being interpreted as an interpolation slot.
    - All changes preserve the existing N-quote matching logic and brace-depth tracking for non-unicode-escape braces.
  </behavior>
  <action>
Modify the three lexer callbacks in `writ-parser/src/lexer.rs`:

**1. `raw_string` callback — add delimiter validation:**

After counting `extra_quotes` and computing `total_quotes` and `search_start`, add a newline check:
```rust
// After extra_quotes computation, check that opening is followed by newline
let first_content = search_start;
if first_content >= bytes.len() {
    return false;
}
// Opening """ must be followed by \n (possibly \r\n)
match bytes[first_content] {
    b'\n' => {}
    b'\r' if first_content + 1 < bytes.len() && bytes[first_content + 1] == b'\n' => {}
    _ => return false, // opening """ not followed by newline
}
```

When the closing delimiter is found (inside the `if quote_count == total_quotes` block), before returning true, validate that only whitespace precedes the closing quotes on their line:
```rust
// Scan backwards from quote_start to find the preceding newline
let mut j = quote_start;
while j > search_start && bytes[j - 1] != b'\n' {
    j -= 1;
}
// Everything from j..quote_start must be whitespace
let prefix = &bytes[j..quote_start];
if !prefix.iter().all(|&b| b == b' ' || b == b'\t' || b == b'\r') {
    return false; // closing """ not on its own line
}
```

**2. `formattable_raw_string` callback — apply same validations:**
Add the same newline-after-opening check and whitespace-before-closing check, adapted for the `formattable_raw_string` callback. The logic is identical.

**3. `formattable_string` callback — handle `\u{` unicode escapes:**

In the `b'\\'` arm, replace the simple `i += 2` with:
```rust
b'\\' => {
    i += 1; // skip the backslash
    if i < bytes.len() {
        if bytes[i] == b'u' && i + 1 < bytes.len() && bytes[i + 1] == b'{' {
            // \u{...} unicode escape — skip past the closing }
            // Do NOT increment brace_depth
            i += 2; // skip 'u' and '{'
            while i < bytes.len() && bytes[i] != b'}' {
                i += 1;
            }
            if i < bytes.len() {
                i += 1; // skip '}'
            }
        } else {
            i += 1; // skip the escaped character
        }
    }
}
```

  <behavior>
    - MarkSweepHeap::new() creates empty heap
    - MarkSweepHeap allocates objects in Vec<Option<HeapObject>> slots, returns HeapRef
    - Freed slots (None) are tracked in a free list and reused on next allocation
    - read_string, get_field, set_field, get_object, get_object_mut work correctly and error on freed slots
    - collect(roots) marks all reachable objects starting from roots using an explicit work-stack
    - Marking traces through HeapObject fields: Struct.fields, Array.elements, Enum.fields, Delegate.target, Boxed inner value
    - Unmarked objects without finalizers are freed (set to None, added to free list)
    - Unmarked objects WITH finalizers are moved to finalization_queue (not freed this cycle)
    - Objects reachable from finalization queue items are kept alive (marked during tracing)
    - finalization_queue is a Vec<HeapRef> of objects awaiting finalization
    - After sweep, finalization_queue items' references are roots for the NEXT collection (two-cycle)
    - collect() returns accurate GcStats: objects_traced (marked count), objects_freed (swept count), heap_before/after, finalization_queue_size
    - mark_finalizable(href: HeapRef) marks an object as having a finalizer
    - drain_finalization_queue() returns and clears the finalization queue (called by runtime after scheduling finalizer tasks)
    - heap_size() returns count of live objects (non-None slots)
    - object_count() returns total allocated slots (including free)
  </behavior>
  <action>
    **Create the MarkSweepHeap struct** in `src/gc.rs` (below the GcHeap trait definition):

    ```rust
    /// Mark-and-sweep garbage collector with finalization support.
    pub struct MarkSweepHeap {
        objects: Vec<Option<HeapObject>>,
        marks: Vec<bool>,
        free_list: Vec<u32>,
        has_finalizer: Vec<bool>,
        finalization_queue: Vec<HeapRef>,
    }
    ```

    **Allocation methods:** When allocating, check free_list first. If non-empty, pop an index, set objects[idx] = Some(obj), clear mark bit and finalizer flag, return HeapRef(idx). If free_list empty, push to end of objects vec, extend marks and has_finalizer vecs.

    **Access methods:** Check that objects[href.0] is Some, otherwise return error. Delegate to the inner HeapObject.

    **collect(roots) algorithm:**

    1. Record `heap_before = self.heap_size()`.
    2. Clear all marks to false.
    3. Create work stack: `Vec<HeapRef>`.
    4. Push all roots onto work stack.
    5. Also push all finalization_queue items onto work stack (their references must stay alive).
    6. **Mark phase:** While work stack is non-empty:
       - Pop an href
       - If href.0 >= objects.len() or marks[href.0] is true, skip
       - Set marks[href.0] = true
       - Get the object, extract all Value::Ref fields, push onto work stack
    7. Count `objects_traced` = number of set marks.
    8. **Sweep phase:** For each slot i where marks[i] is false and objects[i] is Some:
       - If has_finalizer[i] is true:
         - Push HeapRef(i) to finalization_queue
         - has_finalizer[i] = false (will be finalized, clear flag)
         - Do NOT free the object
       - Else:
         - Set objects[i] = None
         - Push i to free_list
         - Increment objects_freed
    9. Record `heap_after = self.heap_size()`.
    10. Return GcStats { objects_traced, objects_freed, heap_before, heap_after, finalization_queue_size: finalization_queue.len() }.

    **Helper for tracing:** Extract all HeapRef values from a HeapObject:
    ```rust
    fn trace_refs(obj: &HeapObject) -> Vec<HeapRef> {
        let mut refs = Vec::new();
        match obj {
            HeapObject::String(_) => {}
            HeapObject::Struct { fields } => {
                for v in fields {
                    if let Value::Ref(href) = v { refs.push(*href); }
                }
            }
            HeapObject::Array { elements, .. } => {
                for v in elements {
                    if let Value::Ref(href) = v { refs.push(*href); }
                }
            }
            HeapObject::Delegate { target, .. } => {
                if let Some(Value::Ref(href)) = target { refs.push(*href); }
            }
            HeapObject::Enum { fields, .. } => {
                for v in fields {
                    if let Value::Ref(href) = v { refs.push(*href); }
                }
            }
            HeapObject::Boxed(v) => {
                if let Value::Ref(href) = v { refs.push(*href); }
            }
        }
        refs
    }
    ```

    **Additional methods:**
    - `mark_finalizable(&mut self, href: HeapRef)` - sets has_finalizer[href.0] = true
    - `drain_finalization_queue(&mut self) -> Vec<HeapRef>` - returns and clears finalization_queue
    - `is_reachable(&self, href: HeapRef) -> bool` - returns marks[href.0] (only valid after collect, before next mutation)

    **Step 1: Add root collection helper** in runtime.rs.

    ```rust
    /// Collect all heap references that are roots for GC.
    fn collect_roots(&self) -> Vec<HeapRef> {
        let mut roots = Vec::new();

        // Task registers
        for task in self.scheduler.tasks.values() {
            for frame in &task.call_stack {
                for reg in &frame.registers {
                    if let Value::Ref(href) = reg {
                        roots.push(*href);
                    }
                }
            }
            // Also check return_value
            if let Some(Value::Ref(href)) = task.return_value {
                roots.push(href);
            }
        }

        // Globals
        for global in &self.scheduler.globals {
            if let Value::Ref(href) = global {
                roots.push(*href);
            }
        }

        // Entity data refs
        for slot in self.scheduler.entity_registry.alive_entities() {
            if let Some(href) = slot.data_ref {
                roots.push(href);
            }
        }

        roots
    }
    ```

    ```rust
    /// when all tasks are quiescent (GC-04).
    pub fn collect_garbage(&mut self) -> GcStats {
        let roots = self.collect_roots();
        let stats = self.heap.collect(&roots);

        // Report to host
        self.host.on_gc_complete(&stats);

        // Schedule finalizer tasks for objects in the finalization queue
        // The heap's finalization queue contains HeapRef values for objects
        // that have finalizers and were unreachable.
        // We need to downcast to MarkSweepHeap to access drain_finalization_queue.
        // Since GcHeap is a trait object, we need a method on the trait for this.
        // Add drain_finalization_queue() to GcHeap trait with a default empty implementation.

        stats
    }
    ```

    Add these to the GcHeap trait in gc.rs with default implementations. BumpHeap's defaults (no-op) are fine. MarkSweepHeap overrides them.

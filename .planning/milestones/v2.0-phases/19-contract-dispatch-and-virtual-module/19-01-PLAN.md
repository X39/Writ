<objective>
Construct the `writ-runtime` virtual module programmatically in memory using the ModuleBuilder API. This module provides all 17 contracts, Option/Result/Range types, primitive pseudo-TypeDefs with intrinsic ImplDef entries, Array<T> with methods and contract implementations, and the Entity base type. No file is read from disk.

Purpose: The virtual module is the foundation for contract dispatch (Plan 03) and cross-module resolution (Plan 02). It populates the metadata tables that the dispatch table consumes. Without it, CALL_VIRT has no contracts or implementations to resolve against.

Output: `writ-runtime/src/virtual_module.rs` with `build_writ_runtime_module() -> Module` function, plus unit tests verifying all metadata table contents.
</objective>

The ModuleBuilder's add_generic_param accepts arbitrary tokens.

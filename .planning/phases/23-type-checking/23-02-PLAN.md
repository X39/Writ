Mutability decisions (from CONTEXT.md):
- Root-binding propagation: `let mut x` -> all fields mutable; `let x` -> all immutable
- `mut self` methods on `let` binding -> error
- Arrays: mutating methods require `let mut`
- For-loop variables: immutable by default
- Dual-span errors: primary = mutation site, secondary = binding declaration

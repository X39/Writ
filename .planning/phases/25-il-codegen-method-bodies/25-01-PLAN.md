  <behavior>
    - TypedExpr::Literal(Int) -> LoadInt; Literal(Float) -> LoadFloat; Literal(Bool) -> LoadTrue/LoadFalse; Literal(String) -> LoadString with string_heap interning
    - TypedExpr::Var -> MOV from locals map register
    - TypedExpr::SelfRef -> MOV from r0 (self is always first param)
    - TypedExpr::Binary with TyKind::Int -> ADD_I/SUB_I/MUL_I/DIV_I/MOD_I; TyKind::Float -> ADD_F etc.; TyKind::Bool EQ -> CMP_EQ_B; TyKind::String + -> STR_CONCAT (stub, EMIT-20 in Plan 04); comparison ops (==, !=, <, >, <=, >=) -> CMP_EQ_*/CMP_LT_* with NOT for inversions
    - TypedExpr::UnaryPrefix Neg -> NEG_I or NEG_F by type; Not -> NOT
    - TypedExpr::If -> emit condition, BR_FALSE to else label, emit then, BR to end label, mark else label, emit else, mark end label
    - TypedExpr::Block -> emit each stmt, emit tail expr if present
    - TypedExpr::Assign -> emit value, emit target (must resolve to register), MOV
    - TypedExpr::Return -> emit value (or void), RET/RET_VOID
    - TypedStmt::Let -> emit value, alloc register, store in locals map, MOV
    - TypedStmt::Expr -> emit expr, discard result register
    - TypedStmt::While -> mark loop_start label, emit condition, BR_FALSE to loop_end, emit body stmts, BR to loop_start, mark loop_end
    - TypedStmt::Return -> emit value, RET/RET_VOID
    - TypedStmt::Break/Continue -> BR to break_label/continue_label (stored in BodyEmitter loop context)
    - TypedStmt::For -> stub with TODO (needs CALL to iterator, deferred to Plan 02 call dispatch)
  </behavior>
  <action>
1. **Create `writ-compiler/src/emit/body/expr.rs`**:
   - `pub fn emit_expr(emitter: &mut BodyEmitter, expr: &TypedExpr) -> u16`: dispatch on TypedExpr variants, return destination register.
   - **Literal emission:**
     - `TypedLiteral::Int(v)` -> alloc reg, emit `LoadInt { r_dst, value: v }`
     - `TypedLiteral::Float(v)` -> alloc reg, emit `LoadFloat { r_dst, value: v }`
     - `TypedLiteral::Bool(true)` -> alloc reg, emit `LoadTrue { r_dst }`
     - `TypedLiteral::Bool(false)` -> alloc reg, emit `LoadFalse { r_dst }`
     - `TypedLiteral::String(s)` -> intern string via `emitter.builder.string_heap`, alloc reg, emit `LoadString { r_dst, string_idx }`
   - **Var:** look up `name` in `emitter.locals`, return that register directly (no MOV needed if just reading)
   - **SelfRef:** return r0 directly
   - **Binary:** emit left -> r_a, emit right -> r_b, alloc r_dst, match on `(op, interner.kind(ty))`:
     - `(Add, Int)` -> AddI, `(Add, Float)` -> AddF, `(Sub, Int)` -> SubI, etc.
     - `(Eq, Int)` -> CmpEqI, `(Eq, Float)` -> CmpEqF, `(Eq, Bool)` -> CmpEqB, `(Eq, String)` -> CmpEqS
     - `(Ne, _)` -> emit Eq variant then `Not { r_dst, r_src: r_dst }`
     - `(Lt, Int)` -> CmpLtI, `(Lt, Float)` -> CmpLtF
     - `(Gt, Int)` -> CmpLtI with r_a/r_b swapped, `(Gt, Float)` -> CmpLtF swapped
     - `(Le, Int)` -> CmpLtI swapped + Not (i.e., `!(b < a)`), or emit CmpLtI + CmpEqI + BitOr
     - `(Ge, Int)` -> emit `CmpLtI { r_dst, r_a, r_b }` then `Not { r_dst, r_src: r_dst }` (i.e., `!(a < b)`)
     - `(And, Bool)` -> BitAnd, `(Or, Bool)` -> BitOr
     - **String `+`:** Emit as placeholder `Nop` with TODO comment — STR_CONCAT handled in Plan 04 (EMIT-20)
   - **UnaryPrefix:** Neg -> NegI/NegF by type. Not -> Not.
   - **If:** emit condition, create else_label + end_label, `BrFalse { r_cond, offset: 0 }` + fixup to else_label, emit then_branch, `Br { offset: 0 }` + fixup to end_label, mark else_label, emit else_branch (or Nop if None), mark end_label. Return r_dst from whichever branch.
   - **Block:** emit each stmt via emit_stmt, emit tail expr if present (return its register), or alloc void register.
   - **Assign:** emit value, resolve target to register (for Var targets, look up locals and MOV; for Field targets, emit as SET_FIELD stub — Plan 02 handles object model).
   - **Return:** emit value expr (if Some), emit `Ret { r_src }` or `RetVoid`.
   - **Path, Error:** Path -> look up in locals like Var. Error -> panic (should never reach codegen per pre-pass).
   - All other variants (Call, Field, ComponentAccess, Index, Match, Lambda, New, ArrayLit, Range, Spawn, SpawnDetached, Join, Cancel, Defer): emit a `Nop` placeholder with a comment noting which plan handles them, and return an allocated void register. This ensures the emitter compiles and handles the full enum without panicking.

2. **Create `writ-compiler/src/emit/body/stmt.rs`**:
   - `pub fn emit_stmt(emitter: &mut BodyEmitter, stmt: &TypedStmt)`: dispatch on TypedStmt variants.
   - **Let:** emit value expr, alloc register (or reuse the value register), insert into `emitter.locals` map.
   - **Expr:** emit expr, discard result.
   - **While:** Save current break/continue labels. Create loop_start/loop_end labels. Mark loop_start. Emit condition, BrFalse to loop_end. Emit body stmts. Br to loop_start. Mark loop_end. Restore break/continue.

   - **Return:** emit value if present, Ret/RetVoid.

   - **Continue:** Br to continue_label.
   - **Atomic:** Placeholder — emit body stmts wrapped with TODO (EMIT-16 in Plan 03).
   - **Error:** panic (should never reach codegen).

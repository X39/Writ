1. **Create `writ-compiler/src/emit/body/patterns.rs`** (EMIT-17, EMIT-23):

   - **`pub fn emit_match(emitter: &mut BodyEmitter, match_expr: &TypedExpr) -> u16`:**
     - Emit scrutinee -> r_scrutinee
     - Determine match kind from scrutinee type:

     **Enum match** (TyKind::Enum):
     ```
     GET_TAG  r_tag, r_scrutinee
     // Create labels: one per variant arm + end label
     SWITCH   r_tag, N, [placeholder offsets]  // fixup each offset
     // For each EnumVariant arm:
     variant_K_label:
       EXTRACT_FIELD r_payload_0, r_scrutinee, 0  // for each binding
       ... insert bindings into locals ...
       emit arm body -> r_arm_result
       MOV r_result, r_arm_result  (if match has result type)
       BR end_label
     // Wildcard arm (if present) — placed after all variant arms:
     wildcard_label:
       emit arm body
       MOV r_result, r_arm_result
     end_label:
     ```
     - SWITCH offsets: for each variant tag 0..N-1, the offset points to the corresponding variant arm label. If a variant has no explicit arm, point to the wildcard arm. Use label fixups: add_fixup for each SWITCH table entry.
     - **Pitfall 5 from RESEARCH.md:** Wildcard arm must handle the fallthrough case. Place it last; SWITCH entries for unmatched variants point to the wildcard label.

     **Option/Result match — ?/try desugared patterns** (EMIT-23):
     - The type checker desugared `expr?` into a Match on Option with two arms:
       - `Some(val)` arm: body = val (unwrap)
       - `None` arm: body = Return(None)
     - Detect this pattern: scrutinee is `TyKind::Option`, one arm returns early.
     - Emit optimized sequence:
       ```
       IS_NONE  r_is_none, r_scrutinee
       BR_FALSE r_is_none, +skip_return
       WRAP_NONE r_none_val   (or LOAD_NULL)
       RET      r_none_val
       skip_return:
       UNWRAP   r_unwrapped, r_scrutinee
       ```
     - Similarly for `try expr` on Result:
       ```
       IS_ERR   r_is_err, r_scrutinee
       BR_FALSE r_is_err, +skip_return
       EXTRACT_ERR r_err_val, r_scrutinee
       WRAP_ERR    r_wrapped_err, r_err_val
       RET         r_wrapped_err
       skip_return:
       UNWRAP_OK   r_ok_val, r_scrutinee
       ```
     - Detection heuristic: Match with exactly 2 arms where one arm's body is `TypedExpr::Return`.

3. **Update `writ-compiler/src/emit/mod.rs`** — restructure `emit()` to produce `Vec<u8>`:
   ```rust
   pub fn emit(
       typed_ast: &TypedAst,
       asts: &[(FileId, &Ast)],
       interner: &TyInterner,
   ) -> Result<Vec<u8>, Vec<Diagnostic>> {
       let mut diags = Vec::new();

       // Pre-pass: abort if any Error nodes exist
       if body::has_error_nodes(typed_ast) {
           diags.push(Diagnostic::error("Cannot emit IL: typed AST contains error nodes"));
           return Err(diags);
       }

       let mut builder = ModuleBuilder::new();

       // Phase 24 metadata collection
       collect::collect_defs(typed_ast, asts, interner, &mut builder, &mut diags);
       slots::assign_vtable_slots(&mut builder);

       // Phase 25: register closure TypeDefs before finalize
       let lambda_infos = body::closure::pre_scan_lambdas(typed_ast, interner, &mut builder);

       builder.finalize();
       collect::collect_post_finalize(typed_ast, asts, &mut builder);

       // Phase 25: emit all method bodies
       let (bodies, body_diags) = body::emit_all_bodies(typed_ast, interner, &builder, &lambda_infos);
       diags.extend(body_diags);

       if !diags.is_empty() {
           return Err(diags);
       }

       // Serialize to binary
       match serialize::serialize(&builder, &bodies, interner) {
           Ok(bytes) => Ok(bytes),
           Err(e) => {
               diags.push(Diagnostic::error(format!("Serialization failed: {}", e)));
               Err(diags)
           }
       }
   }
   ```

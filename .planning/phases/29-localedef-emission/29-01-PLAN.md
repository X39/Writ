<objective>
Implement LocaleDef table emission so the compiler populates table 18 (LocaleDef) for all locale-override `dlg` functions, closing EMIT-25 — the last open requirement in v3.0.

Purpose: LocaleDef rows enable runtime locale dispatch: when the runtime calls a `dlg`, it checks the LocaleDef table for a locale-specific override before falling through to the default version. Without these rows, structural locale overrides are invisible to the runtime.

Output: Modified lowering (attribute passthrough + name suffixing), `collect_locale_defs()` implementation, unit test, E2E test.
</objective>

**2. `writ-compiler/src/emit/collect.rs` — Implement `collect_locale_defs()`**

a) Add a new function `collect_locale_defs`:
```rust
fn collect_locale_defs(typed_ast: &TypedAst, asts: &[(FileId, &Ast)], builder: &mut ModuleBuilder) {
    let def_map = &typed_ast.def_map;

    for decl in &typed_ast.decls {
        let def_id = match decl {
            TypedDecl::Fn { def_id, .. } => *def_id,
            _ => continue,
        };

        let entry = def_map.get_entry(def_id);
        let attrs = find_attrs_for_entry(asts, entry);

        // Look for [Locale("tag")] attribute
        let locale_tag = attrs.iter().find_map(|a| {
            if a.name != "Locale" { return None; }
            a.args.iter().find_map(|arg| {
                if let crate::ast::expr::AstExpr::StringLit { value, .. } = arg {
                    match arg {
                        crate::ast::decl::AstAttributeArg::Positional(
                            crate::ast::expr::AstExpr::StringLit { value, .. }
                        ) => Some(value.clone()),
                        _ => None,
                    }
                } else {
                    None
                }
            })
        });

        let tag = match locale_tag {
            Some(t) => t,
            None => continue,
        };

        // This is a locale override. Its name is "baseName$tag".
        // Extract the base name by stripping the "$tag" suffix.
        let base_name = entry.name.split('$').next().unwrap_or(&entry.name);

        // Look up the base dlg's MethodDef token by its name.
        let base_token = builder
            .methoddef_token_by_name(base_name)
            .map(MetadataToken);

        // Look up this override's MethodDef token.
        let loc_method_token = builder.token_for_def(def_id);

        if let (Some(base), Some(loc)) = (base_token, loc_method_token) {
            builder.add_locale_def(base, &tag, loc);
        }
    }
}
```

IMPORTANT: The actual attribute arg matching must use `AstAttributeArg::Positional(AstExpr::StringLit { value, .. })`. The snippet above is pseudo-code — make sure the actual code uses the correct enum destructuring for `AstAttributeArg`. Import `AstAttributeArg` from `crate::ast::decl` if needed.

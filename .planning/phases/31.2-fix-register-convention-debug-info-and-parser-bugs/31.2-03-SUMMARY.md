## Files Created/Modified

- `writ-compiler/src/emit/body/mod.rs` - Added debug_locals push in both Fn and Impl parameter pre-allocation blocks; name.clone() added to avoid move out of iterator

## Decisions Made
- build_debug_locals takes `&mut builder.string_heap` as a field borrow — Rust allows this since blob_heap (also mutably borrowed in the same loop body) is a different field. No restructuring needed.
- compute_instr_byte_starts is a standalone helper rather than exposing pass-1 data from encode_instructions, keeping encode_instructions self-contained.
- source_spans vec remains empty in practice (body emitter does not yet thread spans through emit_expr calls); this is correct — the fix ensures IF spans were present they would carry correct byte offsets.
- u32::MAX sentinel for end_pc avoids needing scope-exit detection during body emission; clamped to total_code_size at serialize time.

# Writ Scripting Language
A game scripting language with first-class dialogue support, C-style scripting, a Rust-inspired type system, and an entity-component architecture.

## Overview

Writ is designed specifically for game development, combining familiar scripting syntax with powerful features for interactive storytelling and gameplay logic.

## Key Goals

- **First-class Dialogue Support** - Native constructs for writing and managing game dialogue
- **C-style Syntax** - Familiar scripting syntax for easy adoption
- **Rust-inspired Type System** - Strong typing with modern safety features
- **Entity-Component Architecture** - Built-in support for ECS patterns common in game development

## Project Structure

This repository contains the Writ language implementation:

- `writ-parser` - Language parser
- `writ-compiler` - Compiler backend
- `writ-runtime` - Runtime environment
- `writ-cli` - Command-line interface

## Status

Writ is currently in early development. See the [language specification](https://docs.google.com/document/d/1xpKJ9rfnodbr-nSn16wBAWlRkZzGjkRr/edit) for more details.
